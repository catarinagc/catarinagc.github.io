Roll The Credits

Details:

This was a project done for my Master's course Game Desing

Play the Game:

https://an1ratak.itch.io/roll-the-credits

Description:

For this project we were given the theme “The game after the game”. From this, we created Roll The Credits, a 2D platformer where the real challenge begins once the game is supposedly over. The player starts in a classic platforming level, quickly reaches the “end,” and is then dropped into the game’s credits, which turn out to be the actual level. As the player descends through the credits, they must dodge text-based obstacles, traps, and enemies while keeping ahead of the scrolling screen. This project was developed from scratch following a design sprint process, going through all stages from concept to prototype.
